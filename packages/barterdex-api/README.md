### Prerequisites

Following are the minimum tested versions for the tools and libraries you need for running Barterdex API:

- Nodejs: v10.13.0 or newer

- Yarn: v1.9.4 or newer

- Npm: v6.3.0 or newer

- Typescript: v3.2.2 or newer

- TSlint: v5.11.0 or newer
